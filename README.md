#  Motorista - Dashboard Web

Aplicação web desenvolvida para motoristas, com foco em visualização de métricas, controle de viagens e interface moderna estilo SaaS.

O projeto simula um painel administrativo que pode futuramente ser integrado a um backend real com autenticação, controle de rotas e dados em tempo real.

---

## 🌍 Acesse a aplicação

🔗 **Deploy na Vercel:**  
https://front-dash-board-motorista-t4zz.vercel.app/


---

## 🖥️ Funcionalidades

- 📊 Dashboard com métricas
- 🚘 Status de viagens (Em andamento / Concluído)
- 🎨 Interface moderna estilo SaaS
- 🧩 Componentização reutilizável
- 🗺️ Integração com Google Maps (estrutura preparada)
